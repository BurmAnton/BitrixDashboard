Документация REST API получение объекта

                    


                    
                    

                        
Эндпоинт

                        
GET /contacts/api/get/organization/


                        
Назначение

                        
Получение детальной информации об организации по её ИНН.


                        
Аутентификация

                        
Требуется токен в заголовке Authorization.

                        Формат: Authorization: Token <ваш_токен>

                        
Токен можно получить в админ-панели Django (раздел «Токены»). Доступ только у суперпользователей.


                        
Параметры запроса

                        

                        
                        

                            Параметр
                            Обязательный
                            Тип
                            Описание
                        

                        
                        
                        

                            inn
                            да
                            string
                            ИНН организации (10 или 12 цифр)
                        

                        
                        


                        
Примеры запросов


                        
cURL

                        
curl -X GET "https://bitrix24.tuna-edu.ru/contacts/api/get/organization/?inn=7707083893" \
-H "Authorization: Token 4a8389aa7f015d3a68e44e3f4bae94d63c7ac097"


                        
JavaScript (fetch)

                        
fetch('/contacts/api/get/organization/?inn=7707083893', {
headers: {
'Authorization': 'Token 4a8389aa7f015d3a68e44e3f4bae94d63c7ac097'
}
})
.then(response => response.json())
.then(data => console.log(data));


                        
Python (requests)

                        
import requests

headers = {'Authorization': 'Token 4a8389aa7f015d3a68e44e3f4bae94d63c7ac097'}
params = {'inn': '7707083893'}
response = requests.get('https://bitrix24.tuna-edu.ru/contacts/api/get/organization/', 
                headers=headers, params=params)
print(response.json())

Возможные ошибки

                        

                        
                        

                            Код
                            Описание
                            Пример ответа
                        

                        
                        
                        

                            400
                            Не передан ИНН
                            {"error": "ИНН обязателен", "usage": "/contacts/api/get/organization/<ИНН>"}
                        

                        

                            401
                            Отсутствует или неверный токен
                            {"detail": "Authentication credentials were not provided."}
                        

                        

                            403
                            Недостаточно прав
                            {"detail": "You do not have permission to perform this action."}
                        

                        

                            404
                            Организация не найдена
                            {"error": "Организация с ИНН 1234567890 не найдена"}
                        

                        
                        


                        
Примечания

                        
    
                        
  • ИНН должен быть строкой, содержащей только цифры.
    • 
                        
  • Поле projects может быть пустым массивом, если у организации нет связанных проектов.
    • 
                        
  • Поле fed_district вычисляется автоматически на основе региона.
    •
